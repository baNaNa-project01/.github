## 🤝 기여 방법
### **시작하기**
1. 관심 있는 프로젝트를 포크(fork)하고 클론합니다.
   ```bash
   git clone https://github.com/[OrganizationName]/[ProjectName].git
   ```
2. 필요한 패키지를 설치하고 프로젝트를 실행하세요.
   ```bash
   npm install
   npm run dev
   ```
3. 브라우저에서 로컬 서버를 확인하세요: `http://localhost:3000`
